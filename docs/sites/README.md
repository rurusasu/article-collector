# サイトドキュメント索引

このディレクトリの各ファイルは、登録済み site または source 的に扱う site entry を 1 つずつ説明する。これらの docs では、理想形の語彙を使う。

- `Site`: 情報 source。
- `Discovery endpoint`: 記事候補を列挙する feed、API、または page。
- `Fetch route`: 1 件の記事または content URL を取得する経路。
- `Save type`: Markdown 書き出し時に使う content category。

実装は [../architectures/files.md](../architectures/files.md)、[../architectures/enums.md](../architectures/enums.md)、[../architectures/usage.md](../architectures/usage.md) と各 site doc の規約に従う。
